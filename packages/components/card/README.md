# @tera-ui/card

Card is a container for text, photos, and actions in the context of a single subject.

Please refer to the [documentation](https://nextui.org/docs/components/card) for more information.

## Installation

```sh
yarn add @tera-ui/card
# or
npm i @tera-ui/card
```
