# @tera-ui/spacer

Spacer is a component used to add space between components.

Please refer to the [documentation](https://nextui.org/docs/components/spacer) for more information.

## Installation

```sh
yarn add @tera-ui/spacer
# or
npm i @tera-ui/spacer
```
