# @tera-ui/ripple

This is a component that creates a ripple effect on the screen.

Please refer to the [documentation](https://nextui.org/docs/components/button) for more information.

## Installation

```sh
yarn add @tera-ui/ripple
# or
npm i @tera-ui/ripple
```
