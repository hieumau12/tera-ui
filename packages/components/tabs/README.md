# @tera-ui/tabs

Tabs organize content into multiple sections and allow users to navigate between them.

Please refer to the [documentation](https://nextui.org/docs/components/tabs) for more information.

## Installation

```sh
yarn add @tera-ui/tabs
# or
npm i @tera-ui/tabs
```
